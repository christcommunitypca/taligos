# Taligos Apps Website

Simple static site for Taligos Apps.

## Files
- `index.html` — app landing page
- `support.html` — support hub for One Up and Taligo
- `contact.html` — general contact page
- `styles.css` — styling
- `script.js` — mobile nav + email form handling

## Deploy on Cloudflare Pages
Upload the root-level zip so these files are at the top level.
